Macron is a framework for building cutting-edge cross-platform desktop apps using plain HTML, CSS and JavaScript! Learn more.

Features

Blazing Fast

Reach speeds at native performance. Since all resources are at your disposal, all the heavy-lifting is done by Macron.

Native Extensions

For extra performance, you could always take your apps to the next level with compiled C/C++ or Python.

Light & Simple

Macron creates build that are super light and getting started is just as easy as running a few commands. Sit back and watch the speeds!

Cross-platform

Write once, deploy to any platform. Macron currently supports Windows only. Check out our support list.

System Web Engine

Macron reduces build size by over 70% since the native web engine is targeted instead of bundling an entire browser with your app.

Native Installers and Menus

Macron uses truly native menus and installers to give your apps a desktop feeling.

Getting Started!

Would you like to have some fun seeing your app running on Macron? Dead easy. You are a few commands away. 

# Install Macron globally from npm
$ npm install -g macron

# Create new app
$ macron init  MyCoolApp

# Integrate Macron in existing project
$ macron init 

# Remotely run your app
$ macron start

# Build your awesome app
$ macron build

# Run your awesomely built app
$ macron start build
# Requirements (Windows)
- WPF (.NET) version: >=3.5/4.0
- Node version: >=8.11.3
- Python version: >=3.6
- pip version: >= 9.0.1
- pythonnet version: >=2.3.0
- pyinstaller version: >=3.4
- pygubu
- winshell
